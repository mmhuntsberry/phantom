# Style and Conventions
- UI: CSS modules + global CSS tokens in `apps/author/app/global.css`.
- Reuse existing primitives in `apps/author/components` (button/input/label).
- Keep layout shell in `apps/author/app/(site)/layout.tsx`.
- No new styling frameworks; follow existing spacing/typography tokens (`--fs-*`, `--space-*`).
- App Router pages under `apps/author/app/(site)`.
